    container_name=xxx
pid=$(docker inspect -f '{{.State.Pid}}' "$container_name")
sudo mkdir -p /var/run/netns
sudo ln -sf /proc/$pid/ns/net "/var/run/netns/$container_name"
sudo ip netns
sudo ip netns exec "$container_name" ip a
    • netadmin Docker containers do not have full privileges by default. Try adding this to the docker run command:
